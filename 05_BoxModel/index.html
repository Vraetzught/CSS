CSS eigenschappen

Met de selector bepaal je op welke elementen je stijldeclaraties van toepassing zijn. Nu gaan we dieper in op de stijldeclaraties. Welke eigenschappen kan je instellen?

We beginnen met de volgende drie eigenschappen rond lettertype:

  • Font-family
  • Font-style
  • Font-weight

voorbeeld white-space

Met de eigenschap white-space bepaal je hoe je omgaat met spaties en nieuwe lijnen. Is de zin te lang? De helft van deze zin staat op een nieuwe regel.
