Selectors

Een stijlregel bestaat altijd uit een selector gevolgd door één of meerdere stijldeclaraties. Met de selector bepaal je op welke elementen je opmaakregels van toepassing zijn.

We kijken naar de volgende selectors:

  • Element selector
  • Class selector
  • ID selector
  • Pseudoclass selector

Gebruik van spaties

Er mogen spaties staan tussen de selector en de openingsaccolade. Opeenvolgende spaties worden gezien als 1 spatie. h1 { font-family:sans-serif; }
